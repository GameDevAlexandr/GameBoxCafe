# GameBoxCafe

Разработка: Примак Александр Владимирович.

Правила игры:
 Вам необходимо в течении 4 минут набрать максимальное количество очков. 
В начале игры следуйте к грядке, возьмите появившийся ящик с овощами и отнесите его в кафе. 
1 ящик с овощами дает 2 единицы овощей, и каждой единицы овощей производится 1 единица еды и 5 единиц загрязнения.
Покупатели приходя в кафе приобретают еду и идут к лавке, по дороге выбрасывают мусор.
Если загрязнение меньше 50%, покупатели с шансом 50% вернуться в кафе за очередной покупкой.
Мусор необходимо подобрать и отнести на отведенное место (мусорку), иначе по истечению 10 секунд он разложиться и добавит 10 единиц загрязнения.
При накоплении на мусорке 5 единиц мусора, рядом появляется мусорный мешок, который подбирает регулярно проезжающая мусорная машина, за каждый мешок отнимется 5 монет.
Мусорку можно улучшать 2 раза. После первого улучшения покупатели сами будут относит мусор в мусорные баки, после второго стоимость уборки мусорного мешка снизится до 2х монет.
Кафе так-же можно улучшить, тогда при производстве еды оно не будет добавлять загрязнение.

Основные компоненты:

PathFinder - класс осуществляющий поиск кратчайшего пути по дороге к заданной точке.

DataManager - содержит всю информацию о текущем состоянии игры (монеты, загрязнение, и т.д.) и события об изменениях различных параметров.

CharacterBase - базовый класс для Player, Client и TrashCar. Основные параметры: скорость передвижения, позиция для размещения переносимого объекта.

Player - класс осуществляющий управление главным персонажем.

Client - класс обрабатывающий логику поведения клиента.Основные параметры: промежуток времени в который еда после покупки превращается в мусор.

ClientSpawner - генерирует клиентов в заданных точках дороги, реализует паттерн Object Pool. Основные параметры: перерыв между спауном, точки спауна.

StationaryBase - базовый класс для всех стационарных объектов.

Cafe - класс реализующий логику кафе. Основные параметры: время приготовления еды, загрязнение в результате приготовления еды, стоимость еды, количество еды с одного ящика овощей, стоимость улучшения, максимальное количество клиентов в очереди, еды и овощей на складе (задается количеством иконок).

Garden - класс реализующий логику грядки. Основные параметры: время производства, максимальное количество ящиков с овощами на складе (задается количеством иконок).

Bench - класс реализующий логику скамейки. Основные параметры: время отдыха, максимальное количество людей на скамье (задается количеством иконок).

TrashCan - класс реализующий логику мусорки. Основные параметры: количество мусора для создания мешка (задается количеством иконок), стоимость улучшения и цена уборки мешка после улучшения.

ItemBase - базовый класс для всех следующих далее классов. Основные параметры: дальность падения от точки выброса.

Trash - класс обрабатывающий логику мусора, является базовым для Food, TrashBag. Основные параметры: скорость разложения, производимое в результате разложения загрязнение.
